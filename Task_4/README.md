# Farmly – Interactive Prototype

## Overview

This project focuses on converting static UI screens into an interactive user experience using Figma Prototype.

The objective was to simulate how users would navigate through the Farmly grocery application, from launching the app to browsing products, viewing product details, and adding products to the cart.

---

## Project Goal

The goal of this task was to understand user flow design and create meaningful interactions between screens instead of designing isolated interfaces.

The prototype was created to demonstrate how a real user would interact with the application before development begins.

---

## Screens Included

- Splash Screen
- Product Listing Screen
- Product Detail Screen
- Add to Cart Success State

---

## Prototype Flow

1. User opens the app and lands on the Splash Screen.
2. Splash Screen automatically navigates to the Home Screen.
3. User selects a product from the product listing.
4. Product Detail Screen opens with complete product information.
5. User adds the product to the cart.
6. A success state appears confirming the action.
7. User can continue shopping or proceed to the cart.
